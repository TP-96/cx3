About CX‑3 CAD

CX‑3 CAD — a browser-based parametric 3D modeler inspired by SolidWorks.

Built with Three.js. Supports sketching on planes, extrude boss/cut, and STL export.

Keyboard shortcuts:
Ctrl+N New Part · Ctrl+S Save · Ctrl+O Open
L Line · R Rect · C Circle · Esc Cancel
Delete Remove · F Fit All
